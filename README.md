# I_am_On

A bilingual, content-driven Astro starter for a personal site, selected work, writing, and booking.

## What this template is for

- personal introductions
- portfolio / case studies
- writing or notes
- booking / consultation landing pages
- recruiter-friendly summary pages

## Core idea

Content lives in text files. Update Markdown or JSON content, commit, and your site rebuilds with the new copy.

## Suggested next steps

1. Install dependencies with `npm install`
2. Run locally with `npm run dev`
3. Edit files in `src/content/`
4. Deploy with GitHub + Cloudflare Pages
